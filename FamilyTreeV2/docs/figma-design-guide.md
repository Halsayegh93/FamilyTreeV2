# Figma Design Guide — تطبيق عائلة المحمدعلي
## Bold & Colorful Theme | Blue + Silver

---

## 1. إعداد المشروع في Figma

### Frame Size
- iPhone 15 Pro: **393 x 852 px**
- استخدم Auto Layout في كل العناصر

### Grid System
- Columns: 2 (للكروت)
- Margin: 16px
- Gutter: 12px

## 7. المكونات (Components)

### 7.1 الزر الرئيسي (Primary Button)
```
العرض: Full width
الارتفاع: 54px
الخلفية: Gradient Primary (#2C8AF9 → #6C5CE7)
لون النص: #FFFFFF
الخط: Headline (16px Bold)
الزوايا: 16px
الظل: Glow
```

### 7.2 الزر الثانوي (Secondary Button)
```
العرض: Full width
الارتفاع: 50px
الخلفية: Primary 8% opacity
الحدود: Primary 30% opacity, 1.5px
لون النص: Primary
الخط: Callout Bold (14px Semibold)
الزوايا: 16px
```

### 7.3 الكرت (Card)
```
الخلفية: Surface (أبيض / #1A1A2E)
الزوايا: 20px
الظل: Card
الـ padding: 16px
```

### 7.4 كرت التدرج (Gradient Card)
```
الخلفية: أي تدرج من القائمة
الزوايا: 20px
الظل: Card
لون النص: أبيض
```

### 7.5 حقل الإدخال (Text Field)
```
الارتفاع: ~52px
الخلفية: Surface
الزوايا: 12px
الحدود: Gray 12% opacity, 1px
Padding أفقي: 16px
الأيقونة: 44x44px مع خلفية ملونة 12%
```

### 7.6 الأيقونة (Icon)
```
الحجم العادي: 44x44px
الحجم الصغير: 36x36px
الحجم الكبير: 56x56px
الخلفية: لون الأيقونة 12% opacity
الزوايا: 8px
حجم الأيقونة: 16px (عادي)
```

### 7.7 صف الإجراء (Action Row)
```
الارتفاع: ~52px
الأيقونة يسار (44x44)
العنوان: Callout Bold
وصف اختياري: Caption 1
Badge أحمر اختياري
سهم يمين: chevron.right
```

### 7.8 الزر العائم (FAB)
```
الخلفية: Gradient Primary
لون النص: أبيض
الزوايا: Capsule (999px)
الظل: Glow
Padding: 16px أفقي، 12px عمودي
أيقونة: 18px Bold
```

### 7.9 شارة الدور (Role Badge)
```
الخط: Caption 2 Bold
لون النص: لون الدور
الخلفية: لون الدور 12%
Padding: 8px أفقي، 3px عمودي
الزوايا: Capsule
```

### 7.10 بطاقة الإحصاء (Stat Card)
```
العرض: 50% - 8px (عمودين)
الأيقونة: 56x56px
الرقم: Title 1 Black weight
العنوان: Caption 1
الخلفية: Surface
الزوايا: 20px
الظل: Card
```

## 8. تصميم الشاشات

### 8.1 شاشة تسجيل الدخول (Login)
```
الخلفية: Background مع تدرج Primary 8% من الأعلى

العناصر (من الأعلى):
├── Spacer
├── شعار (في المنتصف)
│   ├── دائرة 120x120 - Primary 10% opacity
│   ├── أيقونة tree.fill - 56px - Primary
│   ├── "عائلة المحمدعلي" - Large Title
│   └── "منصة العائلة الرسمية" - Subheadline Secondary
├── Spacer (24px)
├── حقل الهاتف
│   ├── كرت 70px ارتفاع
│   ├── العلم + كود الدولة (100px عرض)
│   ├── فاصل عمودي
│   └── رقم الهاتف - Monospaced 26px
├── زر "طلب رمز التحقق" - Primary Button
├── Spacer
└── "تطبيق خاص لأفراد العائلة" - Caption 2 Tertiary
```

### 8.2 الشاشة الرئيسية (Home)
```
الخلفية: Background

العناصر:
├── Header (تدرج Primary → Purple)
│   ├── صورة المستخدم (48x48 دائرة)
│   ├── "أهلاً [الاسم]" - Title 3 أبيض
│   ├── أيقونة الإشعارات (مع badge)
│   └── الزوايا السفلية: 28px rounded
├── Quick Access Grid (2x2)
│   ├── الشجرة → بنفسجي #6C5CE7 + أيقونة person.3
│   ├── التنبيهات → أحمر #FF6B6B + أيقونة bell
│   ├── الديوانيات → تركوازي #00CEC9 + أيقونة map
│   └── تواصل → ذهبي #FDCB6E + أيقونة phone
├── Section "أخبار العائلة"
├── News Cards (قائمة عمودية)
│   ├── كل كرت يحتوي:
│   ├── صورة المنشور + اسم + الدور + الوقت
│   ├── المحتوى النصي
│   ├── صور (carousel إذا أكثر من واحدة)
│   ├── أزرار: لايك + تعليق + مشاركة
│   └── عداد التعليقات
└── زر عائم "+" لإضافة خبر
```

### 8.3 شاشة الشجرة (Tree)
```
الخلفية: Background مع Grid خفيف

العناصر:
├── شريط البحث (أعلى)
│   ├── حقل البحث مع أيقونة magnifyingglass
│   └── autocomplete dropdown
├── أزرار التحكم (يمين)
│   ├── زر + (تكبير)
│   ├── زر - (تصغير)
│   └── زر Reset
├── الشجرة (ScrollView أفقي + عمودي)
│   ├── العقدة: دائرة 84px
│   │   ├── صورة الشخص
│   │   ├── حدود: Gradient Primary
│   │   ├── الاسم تحت
│   │   └── علامة وفاة إذا متوفي
│   └── خطوط الربط بين العقد
└── إحصائيات أسفل (عدد الأعضاء)
```

### 8.4 شاشة الملف الشخصي (Profile)
```
العناصر:
├── Header (تدرج Primary)
│   ├── صورة المستخدم (100x100 دائرة)
│   ├── حدود متدرجة
│   ├── الاسم - Title 2 أبيض
│   └── شارة الدور
├── بطاقات المعلومات
│   ├── تاريخ الميلاد - DataRow
│   └── رقم الهاتف - DataRow
├── قسم "الأبناء"
│   ├── قائمة الأبناء
│   └── زر "إضافة ابن"
├── الإعدادات - ActionRow
└── زر تسجيل الخروج - Secondary Button (أحمر)
```

### 8.5 شاشة الديوانيات (Diwaniyas)
```
العناصر:
├── عنوان "الديوانيات" - Title 1
├── قائمة الكروت
│   ├── كل كرت:
│   ├── اسم الديوانية - Title 3
│   ├── اسم صاحب الديوانية
│   ├── زر الموقع (رابط Maps)
│   └── أيقونة map.fill تركوازي
├── Empty state إذا فارغ
└── زر عائم لإضافة ديوانية
```

### 8.6 لوحة الأدمن (Admin Dashboard)
```
العناصر:
├── Header "لوحة التحكم" - Title 1
├── إحصائيات (Grid 2x2) - StatCards
│   ├── عدد الأعضاء - أزرق
│   ├── طلبات الانضمام - بنفسجي
│   ├── أخبار معلقة - برتقالي
│   └── الأعضاء النشطين - أخضر
├── قسم "الطلبات" - ActionRows
│   ├── طلبات الانضمام (مع badge)
│   ├── طلبات الأخبار
│   ├── تغيير الهاتف
│   └── طلبات الوفاة
├── قسم "الإدارة" - ActionRows
│   ├── إرسال إشعار
│   ├── قائمة الأعضاء
│   └── التقارير
```

## 9. تصميم أيقونة التطبيق (App Icon)

### المواصفات:
```
الحجم: 1024x1024px
الزوايا: iOS يقص تلقائياً

التصميم:
├── خلفية: Gradient Primary (#2C8AF9 → #6C5CE7)
├── أيقونة شجرة/عائلة في المنتصف
│   ├── لون: أبيض
│   ├── ستايل: SF Symbol "figure.2.and.child.holdinghands"
│   │   أو "tree.fill" أو تصميم مخصص
│   └── الحجم: ~60% من المساحة
└── ظل خفيف داخلي للعمق
```

### البديل:
```
├── خلفية: Gradient (#2C8AF9 → #6C5CE7)
├── حرف "م" (من المحمدعلي) بخط عربي
│   ├── لون: أبيض
│   ├── الخط: SF Pro Rounded Black أو خط عربي مزخرف
│   └── الحجم: كبير في المنتصف
└── دائرة بيضاء شفافة 5% حول الحرف
```

## 11. نصائح التصميم

1. **استخدم Auto Layout** في كل العناصر
2. **أنشئ Components** لكل مكون مذكور
3. **أنشئ Variants** لكل مكون (Light/Dark, Active/Disabled)
4. **استخدم Local Styles** للألوان والخطوط
5. **Layout: RTL** - التصميم من اليمين لليسار (عربي)
6. **أضف صفحة للأيقونات** مع كل SF Symbols المستخدمة
7. **Status Bar**: iOS default (فاتح في الهيدرات المتدرجة)
